Web-Server Installation

Global Bans can be installed on any linux server that supports Go. The application uses MongoDB to store data.

  1. Download the latest release from the releases page
  2. Install MongoDB
  3. Create your MongoDB Database and User with permissions for that database.
  4. Set the following environment variables:
    • MONGO_URI - The URI to your MongoDB database
    • MONGO_DB - The name of your MongoDB database
    • MONGO_USER - The username for your MongoDB database
    • MONGO_PASS - The password for your MongoDB database
    • BASE_URL - The domain the application will run on
    • LOG_FILE - The path to the log file. We recommend using /var/log/globalbans.log
  5. Extract the archive
  6. Run the application with the following command: ./global-bans Or create a service using systemd, or equivalent

Minecraft Plugin Installation

Global Bans uses a plugin to communicate with your Minecraft server. The plugin is available for Spigot servers.

  1. Download the latest release from
  2. Place the plugin in your server's plugin directory
  3. Restart your server
  4. Set the following configuration options in the plugin's config.yml:
    • url - The URL to your Global Bans web server
    • apikey - The API key generated by the web server
